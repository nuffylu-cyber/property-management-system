## 🚀 如何使用

### 快速启动

```bash
# 1. 进入项目目录
cd "D:\claude code\物业管理系统"

# 2. 激活虚拟环境
venv\Scripts\activate

# 3. 运行服务器
python manage.py runserver

# 4. 访问管理后台
# http://localhost:8000/admin/
```

### 访问页面

| 页面 | URL | 功能 |
|------|-----|------|
| 仪表盘 | http://localhost:8000/admin/ | 数据概览 |

| 房产管理 | http://localhost:8000/admin/property/ | 房产、业主、租户 |
